# 🚀 Installation

```bash
pip install ifsclookup
```

---

# ⚡ Quick Example

```python
from ifsclookup import lookup

result = lookup("SBIN0005943")
print(result)
```

# 📦 Features

* ✅ Fast IFSC lookup
* 🏦 Indian bank support
* ⚡ Lightweight package
* 🐍 Beginner friendly
* 📡 Simple API usage
